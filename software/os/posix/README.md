# Portable Operating System Interface (POSIX)
- bench on Unix
- maintainence led by members from IEEE
## Components
It consists of
- Base Definitions
  -	基础数据类型、常量、头文件等
- System Interfaces
  - 系统调用接口，如文件操作、进程管理等
- [Shell](./shell.md) and [Utilities](./Utilities.md)
  - 命令行工具和 shell 行为规范
- Rationale
  - 标准背后的设计理念和解释
