企业架构=业务架构+IT架构
### 业务架构
业务架构是把企业的业务战略转化为日常运作(ops)的渠道

包括业务的
- 运营模式
- 流程体系
- 组织结构
- 地域分布
### IT架构
指导IT投资和设计决策的IT框架
包括
- 信息架构
  - 数据架构
  - 应用架构
- 技术架构
  - 平台服务
